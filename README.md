# MNIST Digit Predictor

A handwritten digit recognition application built with TensorFlow and Tkinter. Draw a digit on the canvas and the model predicts it in real time, complete with a confidence score.

---

## Project Structure

```
proj/
├── model/
│   └── mnist_model.keras   # Saved trained model (generated by train.py)
├── src/
│   ├── train.py            # Model training script
│   └── main.py             # GUI application
└── requirements.txt
```

---

## Requirements

- Python 3.9+
- Dependencies listed in `requirements.txt`

Install all dependencies with:

```bash
pip install -r requirements.txt
```

---

## Usage

### 1. Train the Model

Run this once to train the neural network and save it to `model/mnist_model.keras`:

```bash
python src/train.py
```

Training configuration:

| Parameter     | Value |
|---------------|-------|
| Epochs        | 15    |
| Batch size    | 256   |
| Learning rate | 0.001 |
| Optimiser     | Adam  |

The script loads the MNIST dataset, splits off a 5 000-sample validation set, trains a dense neural network, evaluates it on the test set, and prints the final loss and accuracy.

### 2. Run the Application

```bash
python src/main.py
```

This opens a 280×280 drawing canvas. Draw any digit (0–9) with your mouse and the model predicts it live as you draw. A confidence bar is displayed beneath the prediction. Click **Clear** to reset the canvas.

---

## Model Architecture

A fully connected (dense) sequential network operating on flattened 28×28 greyscale images:

```
Flatten → Dense(512, ReLU) → Dense(128, ReLU) → Dense(10, Softmax)
```

Loss function: sparse categorical cross-entropy.

---

## Notes

- The canvas renders at 280×280 pixels but internally stores drawings at 28×28 (the native MNIST resolution) to match the model's input shape.
- A pre-trained model (`mnist_model.keras`) is included in the repository, so training is optional if you wish to run the GUI straight away.
